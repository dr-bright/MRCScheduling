# MRCScheduling
Scheduling Environment for Multi-Robot Coordination Problems

The repo hosts the code used in the coding tutorial presented at the International Conference on Autonomous Agents and Multi-Agent Systems (AAMAS) 2022 - [Graph Neural Networks: Learning Representations of Robot Team Coordination Problems](https://core-robotics.gatech.edu/2022/01/18/aamas2022_tutorial_gnn_robot/)
